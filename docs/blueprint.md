# **App Name**: VisiFlow

## Core Features:

- Live Check-In Dashboard: A high-visibility stats panel providing real-time data on today's visitor counts and weekly/monthly trends for at-a-glance monitoring.
- Intelligent Entry Intake: Interactive touch-optimized form with native camera integration for profile photos and a high-precision canvas for digital signatures.
- AI Identity Assistant: A vision tool that uses generative reasoning to extract contact information from photos of IDs or business cards, streamlining the entry process.
- Queue & Check-Out Management: Centralized view of all current visitors with instant status toggles and one-tap checkout to maintain site security.
- Visit History & Logging: Filterable database for comprehensive audit trails, allowing users to search and view past entries by date or individual details.
- Automated Reports & Export: Analytical tool generating daily, weekly, or monthly performance metrics exportable to PDF or CSV.
- Admin Control & Access Management: Role-based access controls and secure user management system using JWT authentication.

## Style Guidelines:

- Primary color: Modern Trust Navy (#2E5BFF) for a reliable and professional aesthetic.
- Background color: Deep Sky Tint (#F1F4FA) for a clean, minimal, and expansive workspace feel.
- Accent color: Precision Cyan (#00C6D1) for interactive UI elements and status indicators.
- Primary and Body font: 'Inter'—a modern, machined sans-serif that provides clarity and an objective, technological feel.
- Clean, thin-line stroke icons from a modern set like Lucide, using the Primary color to denote interactivity.
- Grid-based tablet architecture featuring large tap targets (minimum 48px) and card-based groupings for easy navigation.
- Subtle 200ms ease-in transitions for form state changes and smooth horizontal slides between flow steps.
